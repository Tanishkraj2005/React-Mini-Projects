# Currency Converter

A clean and responsive React currency converter built with Vite.

## Live Demo

- https://currency-converter-08.vercel.app/

## Features

- Convert values between multiple currencies
- Real-time calculation while typing
- Simple and intuitive interface
- Built with React and Vite for fast performance

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/currency-converter.git
   ```
2. Change into the project folder:
   ```bash
   cd currency-converter
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Available Scripts

In the project directory, you can run:

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build locally

## Usage

1. Run `npm run dev`
2. Open the local server URL shown in the terminal
3. Enter an amount and select currencies to convert
4. View the converted result instantly

## Project Structure

- `src/` - application source code
- `public/` - static assets and HTML template
- `package.json` - project scripts and dependencies
